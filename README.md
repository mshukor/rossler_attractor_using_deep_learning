# Rossler Attractor using Deep Learning

### Installation

Please install the following basic packages:
`matplotlib`, `torch`, `numpy`, `pandas`, `tqdm`, `sklearn`, `scipy`.

### Run the training
Run the following command with corresponding arguments:

```
python time_series.py 
```
You can train both the feedforward model (MLP) and the time dependent one (using GRU)

### Run the prediction
Run the following command with corresponding arguments:
```
python train.py
```
